## Quick-start - running this repo localy

### Install deps

On Ubuntu:
```
sudo apt install ruby-full build-essential zlib1g-dev
sudo gem install jekyll bundler
```

On Mac:
```
brew install ruby-dev
gem install jekyll bundler
```

### Installing other deps
```
bundle
```

### Run server
```
jekyll serve
```
